Which Maven files should you commit to your git repository?
- The **`pom.xml`** file
- (Optionally) The Maven wrapper files:
  - **`.mvn`** folder
  - **`mvnw`** file
  - **`mvnw.cmd`** folder

Which should you ignore?
- The **`target`** folder
- Everything else that you generate during build
